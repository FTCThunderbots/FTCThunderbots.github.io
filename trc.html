The Thunderbots Robot Controller

A dynamic jar-loading version of the FTC Android Robot Controller app

The Thunderbots Robot Controller is an improved version of the FTC Android Robot Controller app. It features an improved OpMode loading system, as well as a Thunderbots-themed texture pack. It is fully open source, and is compatible with any existing FTC app code.

Advantages of the jar-loading system include:

  • The ability to use any Integrated Development Environment (IDE) instead of being restricted to Android Studio
  • The removal of the long load time to push code onto the phone

Instead of having to recompile the app into an APK everytime new code is written, Code must be compiled into JARs and put in the FIRST folder of the phone's storage. One limitation of the Android system used is that code must be compiled with JRE 7 or earlier. The existing FTC RobotCore SDK may continue to be used, as can all the other included libraries. Currently, OpModes must be annotated with the @Active annotation in order to show up in the OpMode list of the Driver Station.
