Summer Camps 2019


Curriculum

In this camp, students will have the opportunity to discover the basics of building and programming robots. They will learn how to make a successful robot using LEGOs while developing important real world skills such as teamwork and gracious professionalism. In addition, they will program using software from LEGO Mindstorms. This will give them an understanding of basic coding structures. In the end, students will be able to build and program their own robot to compete with each other on a game field. We hope to see you at this exciting new robotics experience!


There will be an entertaining robotics camp for 5th-8th graders at
Mountain View High School in Room 111 from 9:00 to 3:00 for beginning coders.
